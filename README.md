# 📘 学習進捗トラッカー

参考書や教材の進み具合を管理するシンプルなWebアプリです。  
現在のページ数を入力するだけで、残りページ・完了日・試験までに間に合うかを自動計算します。

---

## 🔧 機能

- 📄 現在のページから残りページを自動計算
- 📅 完了までの日数を算出
- ⏱ 試験日までに間に合うか判定
- 📱 スマホ対応（ブラウザでそのまま使える）

---

## 🚀 使い方

1. 現在のページ数を入力  
2. 1日あたりの進捗ページ数を入力  
3. 試験日を入力  
4. 「計算する」を押す  

→ 自動で進捗状況が表示されます
